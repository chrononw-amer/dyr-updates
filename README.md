# DYR — Real Estate Management System

A professional desktop & mobile real estate management application built for internal company use.

---

## 📦 Latest Release: v1.8.20

> Released: 2026-04-16

### New Features
- **Building Cards — Available Area** — AVAILABLE count now correctly excludes locked units and shows their total area (m²) inline.
- **Building Detail — Inventory Stats Banner** — New colored stats banner at the top of each building page showing Available / Locked / Contracted / Offered with unit count + total m² area for each category.
- **Building Cards — Area Section** — Contracted area shown in green with %, Offered area in amber with % of total building area.
